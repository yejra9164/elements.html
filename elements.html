HTML Elements with Examples

1. Heading Element

The <h1> to <h6> elements are used to define HTML headings.

2. Paragraph Element

The <p> element defines a paragraph in HTML. It helps structure text content.

3. Image Element

The <img> element is used to embed an image in a webpage.

4. Anchor (Link) Element

The <a> element is used to define a hyperlink.
